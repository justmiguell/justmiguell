## SOBRE

Curso **ADS** no IFSP Catanduva e **Engenharia de Software** na Estácio, ao mesmo tempo.
Foco em **arquitetura em camadas** e código comentado, pra qualquer projeto ser fácil de
entender e explorar. Cada projeto de portfólio carrega um tema que eu gosto de verdade —
bandas, música, mundos de fantasia.

## PROJETO EM DESTAQUE

<div align="center">

### 🩸 Deadwax

Catálogo de bandas e álbuns onde as pessoas postam suas músicas favoritas e avaliam
publicamente — Java 17, Spring Boot e Spring Data JPA (relacionamentos entre banda,
álbum e avaliação), front-end em Thymeleaf e banco H2 persistente. Identidade visual
construída do zero.
